## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies:
  `npm install <package> --save`
- Run the development web server:
  `node express_server.js`
- The appropriate URL to access the website on your computer is: localhost:8080
